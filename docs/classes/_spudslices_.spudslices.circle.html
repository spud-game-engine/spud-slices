points

points: any[] = []

Inherited from Shape.points

  • Defined in spudslices.ts:90

Even though (at time of writing) this is type any[] it is one of two things:

  • number[][], given that it is an ordinary point. (See dimensions) for more info.
  • number[], given that it is the first point, and the class type is Circle (or one of it's children). In this case, this point is the radius, and the next point is the center of the circle.
